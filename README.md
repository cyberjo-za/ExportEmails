# Export Email Application 

## What it does
This application allows you to export a users emails on a server if you know the server details and the user's credentials.

## Compatibility
It should only work for email services that support basic auth. Currently it doesn't work with oauth or any special kind of authorisation that is needed. 

## Why this program was created
To allow the export for archival purposes so that email can be closed or cleared without needing to do a whole email setup etc.

## How to use it
It is recommended that you use the basic GUI that it comes with. You can just insert the required details and select the folder location you want the PST file in. The GUI also show progress. 
- Open the exe in the release. 
- Fill in the details. 
- Select your file location. 
- Click export and monitor the progress.
